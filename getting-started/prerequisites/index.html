    Prerequisites

    Clone the repo¶

    git clone https://github.com/ConsenSys/UniversalToken-extendable

    cd UniversalToken-extendable

    Add the libraries¶

    yarn add truffle

yarn add dotenv

yarn add @truffle/hdwallet-provider
